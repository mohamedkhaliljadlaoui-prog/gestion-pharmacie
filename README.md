# Pharmacy Manager – Gestion de Pharmacie

Application Java desktop complète pour la gestion d’une pharmacie.  
Projet universitaire – Architecture MVC, pattern DAO, Swing, JDBC, MySQL.

## ✨ Fonctionnalités principales

- Authentification sécurisée (Pharmacien / Gestionnaire)
- Tableau de bord adapté au rôle
- Gestion des médicaments (CRUD, stock, seuil d’alerte)
- Enregistrement des ventes avec calcul automatique
- Gestion des clients
- Gestion des commandes de réapprovisionnement
- Historique des mouvements de stock
- Alertes de stock critique (visuelles + email)
- Génération de rapports (CSV, HTML)
- Export de données
- Interface moderne avec Java Swing

## 🧱 Architecture technique

- **Langage** : Java SE 11+
- **Interface** : Swing (Look & Feel personnalisé)
- **Base de données** : MySQL
- **Persistance** : JDBC + pattern **DAO**
- **Architecture** : MVC
- **Transactions** : TransactionManager personnalisé
- **Patterns** : Singleton, DAO, MVC
